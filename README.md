# TaskMongo

# API 
Créez un .env avec : 
```
    DATABASE_URL = "mongodb:// 10.79.49.64:27017/taskmongo"
```
démarer le serveur : 

```shell
    npm run start
```

démarer le serveur en dev: 

```shell
    npm run dev
```


# Front

installer angular global : 

```shell
    npm install -g @angular/cli
```

démarer le serveur : 

```shell
    npm run start
```
